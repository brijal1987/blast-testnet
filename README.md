# blast-testnet

## Clone repository
```bash
# development
$ git clone https://github.com/brijal1987/blast-testnet.git

$ cd blast-testnet
```

## installing dependencies
```bash
# development
$ npm install
```

## Running the app
```bash
# development
$ npm start

$ open http://localhost:3000  in browser

OR

$ node Web3.js
```
